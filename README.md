# numeron
java勉強用 nyumeron

## 要件定義

### ユーザー側
* コマンドにキーボードから入力する
-> 4桁の数字を入力してもらう想定

### プログラム側
* ランダムに4桁の数字を生成する。
-> 4桁を構成する数字は重複が無いようにする

* Hit Blowでユーザが推測した数値の結果を返す処理


## 必要機能

* ユーザーがコマンドに数字を入力する
* ランダムに数字を生成する
* ユーザー入力情報のバリデーション
